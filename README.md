## üåü Key Features

- **1000 Curated Terms**: Comprehensive vocabulary tailored for daily clinical practice in the Netherlands.
- **CEFR Level Filtering**: Focus your study on levels A2 (Pre-Intermediate) through C2 (Mastery).
- **Interactive 3D Flashcards**: Sleek, high-performance card flipping animation with persistent 3D transforms.
- **Contextual Learning**: Every card includes:
  - **Dutch Term**: The clinical word or phrase.
  - **English Translation**: Precise medical equivalent.
  - **Explanation**: Detailed context of the term's use.
  - **Example Sentence**: Realistic clinical usage in Dutch.
- **Comprehensive Topic Coverage**:
  - **Clinical Areas**: Cardiology, Neurology, Oncology, Pediatrics, Obstetrics, Geriatrics.
  - **Daily Care**: Wound care, hygiene, vital signs, medication administration.
  - **Infrastructure**: Dutch healthcare system, insurance (Zorgverzekering), and facilities.

## üöÄ Getting Started

### Prerequisites

You only need a modern web browser.

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/vikramrajj/Dutch.git
   cd Dutch
   ```

2. Serve the application using any static server:
   - Using Python: `python -m http.server 3000`
   - Using Node.js: `npx serve .`

3. Open your browser to `http://localhost:3000`.
